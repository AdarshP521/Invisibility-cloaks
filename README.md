# Invisibility-cloaks

This project implements a real-time invisibility cloak effect using Python and computer vision techniques. The system detects a specific color (typically red) and replaces it with the background captured earlier, creating the illusion that the object covered by the cloth is invisible.

The project is inspired by the invisibility cloak shown in Harry Potter and the Philosopher's Stone and is implemented using the powerful computer vision library OpenCV.

🔹 Key Concepts Used:
• Real-time video processing
• Color detection using HSV color space
• Image masking and morphological transformations
• Background replacement using computer vision
